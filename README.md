# neo-maikura-modoki

A tiny voxel-based sandbox game written in Python3 using OpenGL 4.6 and GLFW 3.  
This is fork of [maikura-modoki](https://github.com/Rinwasyu/maikura-modoki) (using OpenGL 1.x).
